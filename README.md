# Manga Hub

An app where the user can search for their favorite anime or manga. The user can create an account or login to an existing account to add/remove favorites.

## Technologies

-  HTML/CSS
-  Javascript
-  React.js
-  Nodejs
-  Bootstrap
-  Create-react-app
-  Apollo
-  Express
-  MongoDB
-  Mongoose
-  Passport
-  GraphQL
